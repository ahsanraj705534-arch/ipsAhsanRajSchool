What This Portal Does

Main functions available for Indian Public School

Student Data Collection

Students or guardians can submit admission and student details directly from the website.

Class-wise Management

Every record is stored with class and section so staff can open the correct list quickly.

Office Record Updates

Admin users can review entries, edit mistakes, delete duplicate records, and keep data clean.

Search and Filters

Records can be filtered by class, section, or student name to save office time.

Print-ready Details

Each student has a separate detail and print page for paperwork and quick reference.

Excel Download

All student entries can be exported into Excel format for reporting and backup work.

User Workflows

Step-by-step use for students and school staff

Student Side

How students or guardians should fill the form

  1. Open the Student Form page from the main menu or class-wise quick access section.
  2. Enter the student details carefully, including name, parents' names, date of birth, gender, class, section, mobile number, and address.
  3. Check that the selected class and section are correct so the record goes into the proper class list.
  4. Submit the form and wait for the success message confirming the entry was saved.
Admin Side

How school staff should manage records

  1. Open Admin Login and sign in with the school office username and password.
  2. Use View Students to open the full dashboard or jump into a specific class list.
  3. Search by student name or filter by class and section to find records quickly.
  4. Open a student record to edit details, print the profile, or delete incorrect entries when needed.
Data and Controls

What gets saved and what admin users can do

Student data fields collected

  • Student Name: Main identity for the record.
  • Father's Name and Mother's Name: Parent information for school records.
  • Date of Birth and Gender: Basic student profile details.
  • Class and Section: Used to place the student in the correct class-wise list.
  • Mobile Number: Contact number with validation for proper format.
  • Address: Home address for official school records.

Admin tools available in the dashboard

  • View all students: Open the complete student table in one place.
  • Class-wise access: Jump directly into a specific class list for easier office work.
  • Edit and delete: Correct mistakes and remove wrong entries when required.
  • Student detail page: Review the full profile before taking action.
  • Print view: Generate a clean paper-friendly version of student information.
  • Download Excel: Export the full record set into spreadsheet format.
Validation

Form accuracy checks

The form checks required fields, student and parent names, date format, class and gender selection, and the 10-digit mobile number format before saving.

Student ID

Automatic record identity

Each saved student receives an auto-generated student ID so records stay unique and easier to trace in the system.

Mobile Use

Responsive design for phones

The portal is designed to work on mobile devices so students, guardians, and office staff can use it comfortably on smaller screens.
